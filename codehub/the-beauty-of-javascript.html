The Beauty of JavaScript

Intro to JavaScript (JS)

JavaScript is a very powerful client-side scripting language that is used mainly for enhancing the interaction of a user with the webpage. That means, you can make your webpage more lively and interactive, with the help of JavaScript. It is also being widely used in game development like this flappy bird game Here and Mobile application development.

The language was called LiveScript and was later renamed JavaScript. There are many programmers who think that JavaScript and Java are the same. In fact, JavaScript and Java are very much unrelated. Java is a very complex programming language whereas JavaScript is only a scripting language. The syntax of JavaScript is mostly influenced by the programming language C.

Example of a JavaScript Program

You should place all your JavaScript code within <script> tags (<script> and </script>) if you are keeping your JavaScript code within the HTML document itself. This helps your browser distinguish your JavaScript code from the rest of the code.


	<html>
		<head>
		<title>My First JavaScript code!!!</title>
			<script type="text/javascript">
				alert("Hello World!");
			</script>
		</head>
		<body>
		</body>
	</html>
